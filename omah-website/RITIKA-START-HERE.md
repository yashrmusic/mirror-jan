# Omah Website Handoff

Live site:
- https://omah-website.vercel.app

Editable files:
- `index.html` - page content and product cards
- `styles.css` - colors, layout, typography, responsive design
- `script.js` - product slider and ritual picker behavior
- `strategy/` - Instagram and UGC planning docs
- `ugc-automation/output/week-01/weekly-content-pack.md` - ready-to-review Reel scripts and captions

Brand direction:
- Brand name: Omah
- Moodline: Begin within.
- Visual reference: Lemme-inspired playful wellness styling
- Tone: soft, calm, modern, lifestyle-first, not heavy spiritual language

How to work with Codex:
1. Open this GitHub repo/folder in a Codex workspace.
2. Ask Codex to edit the specific file or section.
3. Preview locally by opening `index.html`.
4. Push changes back to GitHub.

Current Vercel deployment was done from Yash's local folder, so GitHub edits may need a redeploy unless Vercel is connected to this repo later.

For Instagram/UGC feedback, open `ugc-automation/output/week-01/weekly-content-pack.md` and comment in normal English on what feels right or off.
